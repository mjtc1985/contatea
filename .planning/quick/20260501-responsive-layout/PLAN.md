# Quick Task: Responsive Counting Layout

## Problem
Counting items appear in a single line on wide screens (tablets) because the image size shrinks too much as the number of items increases.

## Solution
- Improve `_calculateImageSize` to provide larger, more consistent sizes.
- Ensure the `Wrap` widget handles multi-line layout effectively.
- Add some padding and constraints to prevent items from stretching too far horizontally.
