# Contatea

Aplicación móvil y tablet diseñada para facilitar el aprendizaje de conceptos básicos (conteo y asociación de palabras) en niños con Trastorno del Espectro Autista (TEA) o discapacidad intelectual.

## Características
- **Juego de Conteo**: Aprendizaje numérico con apoyo visual dinámico.
- **Asociación Imagen-Palabra**: Desarrollo de vocabulario y lectura global.
- **Personalización Total**: Los tutores pueden usar la cámara, la galería o el buscador de ARASAAC para adaptar el contenido.
- **Modo Offline**: Una vez configurada, la app funciona sin conexión a internet.
- **Sin Distracciones**: Interfaz limpia, sin publicidad y con refuerzo positivo sencillo.

## Instalación
Para instalar en Android:
1. Habilitar el modo desarrollador y depuración USB en la tablet.
2. Conectar al PC.
3. Ejecutar `adb install app-release.apk`.
